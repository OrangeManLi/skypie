<h2>这是一个html5小游戏</h2>
<br>
<h6>采用技术：</h6>
<p>
采用2014年国内推出的egret游戏引擎。egret采用typescript语言编写。
</p>
<h6>游戏介绍：</h6>
<p>
游戏名为《天降美食》，控制小怪物吃从天而降的美食。
</p>
